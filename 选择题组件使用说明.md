# 选择题组件使用说明

## 功能概述

已实现简化的选择题功能：

1. **智能识别** - 检测步骤标题是否包含"选择题"字样
2. **组件切换** - 自动使用 `ChoiceQuestion` 组件渲染选择题
3. **默认内容** - 提供示例选择题内容

## 实现逻辑

### 学习路由组件 (`src/views/step/learn/index.vue`)

```typescript
// 检查标题是否包含"选择题"
const isChoiceQuestion = step.title && step.title.includes("选择题");

return isChoiceQuestion ? ChoiceQuestion : ViewMd;
```

### 选择题数据

当检测到标题包含"选择题"时，会使用以下默认数据：

```typescript
{
  questions: "以下代码运行什么？",
  code: `def print_double(x):
    print(2 * x)
print_double(3)`,
  options: ["6", "2", "3", "0"]
}
```
