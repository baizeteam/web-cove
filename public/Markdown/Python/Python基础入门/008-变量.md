## 变量是什么

变量就像一个小盒子，你可以把各种东西放在里面，比如数字、文字。

## 怎么创建变量

给变量取个名字，然后用"="把东西放进去。

比如 age=13, "age"是名字，13就是放在里面的东西。

名字不能以数字开头，并且要有意义。

## 变量能做什么

可以放数字（例如1、3.14），和文字（要用引号包裹起来，例如"小明"），和布尔值（只有 True 和 False）。

## 怎么用变量

可以用变量做计算。

比如有个变量x, 可以写成 x = 23，也可以用 print() 函数把变量的值显示出来。

```python
x = 23
print(x)
```

**多个变量可以指向同一个对象**。

![](https://cdn.nlark.com/yuque/0/2025/png/21914976/1755448547255-9392521b-e3f6-4bab-8422-528cba7b1855.png)

**变量可以进行相关运算**。

![](https://cdn.nlark.com/yuque/0/2025/png/21914976/1755448599842-3caa4b5b-e0ac-4d80-a8e8-ba2266301592.png)

```python
x = 7
x = x + 3
print(x) # 10
```

> 你可以使用变量来执行相应的操作，就像使用数字和字符串一样。正如你所看到的，变量在程序中存储它的值。
