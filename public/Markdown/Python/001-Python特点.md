**易于学习**

Python 有相对较少的关键字，结构简单。

**易于阅读**

Python 代码的定义更清晰。

**丰富的生态**

Python\*\* \*\*的最大优势之一是丰富的跨平台的库，在 Linux ，Windows 和 MacOS 中都有很好的兼容。

**可扩展**

如果你需要一段运行很快的关键代码，你可以使用 C 或 C++ 完成那部分程序，然后从你的 Python 程序中调用。

**数据库**

Python 提供所有主要的商业数据库的接口。
