# Markdown 全语法示例文档

## 一、标题（Headings）
Markdown 支持 6 级标题，使用 `#` 表示，数量对应级别：

# 一级标题（# + 空格）
## 二级标题（## + 空格）
### 三级标题（### + 空格）
#### 四级标题
##### 五级标题
###### 六级标题

## 三、文本格式（Text Formatting）
- **加粗**：`**加粗文本**` → **这是加粗文本**
- *斜体*：`*斜体文本*` 或 `_斜体文本_` → *这是斜体文本*
- ***加粗+斜体***：`***混合文本***` → ***这是混合格式文本***
- ~~删除线~~：`~~删除文本~~` → ~~这是被删除的文本~~
- <u>下划线</u>：HTML 语法）→ <u>这是下划线文本</u>
- **<span style="color:red">print()</span>** 函数可以用来输出文本（高亮文本）。



## 四、列表（Lists）
### 1. 无序列表（Unordered Lists）
使用 `-`、`*` 或 `+` 作为前缀：
- 列表项 1
- 列表项 2
    - 嵌套列表项
    - 嵌套列表项
- 列表项 3

### 2. 有序列表（Ordered Lists）
使用数字 + `.` 作为前缀：
1. 第一步
2. 第二步
    1. 子步骤 
    2. 子步骤 
3. 第三步

### 3. 任务列表（Task Lists）
- [x] 已完成任务（`- [x]`）
- [ ] 未完成任务（`- [ ]`）
